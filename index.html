Les balises

  • Les balises paires

    • Elles s'ouvrent, contiennent du texte, et se ferment plus loin. Exemple : 

     < titre> < /titre> 
     < head> < /head> 
     < p> < /p>

     < ul> < /ul> 
    

     < body> < /body> 
    
    
  • Les balises orphelines
    • 
    
    Ce sont des balises qui servent le plus souvent à insérer un élément à un endroit précis (par exemple une image).
     Il n'est pas nécessaire de délimiter le début et la fin de l'image, on veut juste dire à l'ordinateur « Insère une image ici ».

Exemple :

     < image/> 
    

     < meta charse t="utf-8" /> 
    

     < br />

Les attributs

Les attributs sont un peu les options des balises. Ils viennent les compléter pour donner des informations supplémentaires. L'attribut se place après le nom de la balise ouvrante et a le plus souvent une valeur.balises et attributs"

différence entre inline et block

En HTML, tout élément est soit de type « block » (bloc), soit de type « inline » (en ligne). Block :Occupe toute la largeur disponible tandisque inline N'occupe que la largeur indispensable à l'affichage du contenu et ne provoque pas de retour à la ligne.Exemples de balises block :
